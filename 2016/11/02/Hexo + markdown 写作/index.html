Hexo + markdown 写作

发表于 2016-11-02   |   分类于 Hexo

标题

标题最多有最多6级，使用过程中发现Hexo中使用一级标题时，段后空间过窄，所以实际从二级标题开始使用。

锚点

锚点可以理解为文章中的快速定位，具体使用方法如下

[说明文字](#锚点名字)

<span id = "锚点名字">说明文字</span>

引用

在文章中插入引用段，引用的区块内也可以使用其他的markdown语法。

或使用Hexo Tag Plugins（可包含作者、来源和标题）
